# Objects in JavaScript

Objects are collections of key-value pairs used to store related data and functionality.

## Creating Objects

### Object Literal
`const obj = { key: value };`

### Object Constructor
`const obj = new Object();`

### Constructor Function
Custom object creation pattern

## Accessing Properties

### Dot Notation
`obj.propertyName`

### Bracket Notation
`obj['propertyName']`

## Object Methods

### Built-in Methods
- `Object.keys()` - Get array of keys
- `Object.values()` - Get array of values
- `Object.entries()` - Get array of [key, value] pairs
- `Object.assign()` - Copy properties from source to target
- `Object.freeze()` - Make object immutable
- `Object.seal()` - Prevent adding/removing properties

## Object Concepts

### this Keyword
Refers to the current object

### Methods
Functions stored as object properties

### Prototypes
Objects inherit properties from prototypes

### Destructuring
Extract properties into variables
